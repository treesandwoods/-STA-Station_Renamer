# Dribble-Station-Renamer
OpenTTD Game Script.

**Fork of GingerNinjaXL's -STA-Station_Renamer**

Renames stations how I prefer them.

Rail:   Town  
Bus:    Town S01  
Tram:   Town T01  
Truck:  Town M01  
Air:    Town Airport  
Sea:    Town Dock  

Force a rename by renaming the station "!rename".
